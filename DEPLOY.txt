5. Click "New +" (top right) → "Web Service"
6. Find and click "Connect" next to trabahanap repository
7. Fill in these EXACT settings:

   Build Command: npm install
   
   Start Command: node server.js
   
   Instance Type: Free

8. Click "Create Web Service" (blue button at bottom)
9. Wait 2-3 minutes while it deploys
10. Look for "Live" status with green dot ✅
