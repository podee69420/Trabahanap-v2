## 🚀 How to Deploy

### Quick Steps:

1. **Delete your old Render service** (if exists)
   - Go to Render dashboard
   - Settings → Delete Web Service

2. **Update your GitHub repository**
   - Replace ALL old files with these new ones
   - Make sure folder structure matches above

3. **Create new Render service**
   - Same settings as before:
   - Build Command: `npm install`
   - Start Command: `node server.js`
   - Instance Type: Free

4. **Deploy and test!**

## 🐛 Debugging

If registration doesn't work:
1. Check Render logs for errors
2. Look for: `✅ New worker registered: [Name]`
3. Should also see: `📊 Total workers now: X`

If profiles don't show:
1. Click on a worker card
2. Check browser console (F12)
3. Should see modal popup
